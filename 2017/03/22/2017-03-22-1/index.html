这到底是什么！！！


Todolist 是一款基于 GTD（尽管去做 ／ Getting Things Done）的命令行式任务管理工具

安装

在 macOS 下面利用brew install todolist进行快速安装。

开始

1. 初始化todolist 仓库

运行todolist init初始化本地的.todos.json文件

2. 添加一条 todolist

可以用命令add或者a添加一条事件。

3. 查看列表

用命令list 或者l,查看事件列表.

另外，用命令todolist list due tom 查看明天的事件。
用命令todolist list by project可以按照工程项目进行查看

4. 事件完成标记

利用命令todolist complete 或者 todolist c

完成全部事件

5. 编辑事件

使用todolist e [id] due [due]对事件进行编辑

6. web界面

使用todolist web命令，可以建立一个 todolist 的 web 页面。
